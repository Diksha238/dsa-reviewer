# DSA Reviewer — Frontend

A clean, production-grade Next.js frontend for the DSA Code Reviewer project.

## Setup

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Requirements
- Node.js 18+
- Spring Boot backend running on port 8080

## Pages
- `/` — Login / Register
- `/review` — Main code review page  
- `/history` — Review history
- `/history/[id]` — Review detail

## Tech Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Lucide React icons
- Syne + JetBrains Mono fonts
